#Ved Portfolio

Description:
This is a personal portfolio website built using the iPortfolio template from BootstrapMade. It showcases projects, skills, and personal information in a clean and responsive design.

Live Website:
https://ved-portfolio.onrender.com/

----------------------------------------
How to Run Locally
----------------------------------------

Follow these steps to run the project on your local machine:

1. Open terminal / command prompt
2. Navigate to your project folder:
   cd ved-myPortfolio

3. Start a local server using Python:
   python -m http.server 8000

4. Open your browser and go to:
   http://localhost:8000/

----------------------------------------
Notes
----------------------------------------

- Make sure Python is installed on your system.
- This is a static website, so no build step is required.
- The contact form (PHP-based) will not work on static hosting platforms like Render.
